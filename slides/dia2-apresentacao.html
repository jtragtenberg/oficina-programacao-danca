// sistema vestibular
Labirinto do ouvido interno — detecta aceleração linear e velocidade angular.
O mesmo que o MPU6050 faz em silício.
// acelerômetro

Acelerômetro

Mede aceleração linear nos 3 eixos (X, Y, Z) em g (gravidade = 9,8 m/s²).


Em repouso, o eixo alinhado com a gravidade lê ±1g. Os outros leem ≈0.


Eixos: X = lateral · Y = frente/trás · Z = cima/baixo

// giroscópio

Giroscópio

Mede velocidade angular em graus/segundo. Não mede posição — mede quão rápido está girando.

// fusão de sensores

Madgwick AHRS

// o problema

O giroscópio integra velocidade angular para estimar ângulo — mas acumula erro com o tempo (drift).
O acelerômetro sabe onde é "baixo" (gravidade), mas é ruidoso durante movimento.

// a solução

AHRS — Attitude and Heading Reference System.
Combina os dois: o giroscópio rastreia rotação rápida, o acelerômetro corrige continuamente o eixo vertical.

// madgwick

Algoritmo de fusão eficiente — roda em tempo real no ESP32.
Saída: quaternion (w, x, y, z) que representa a orientação completa no espaço 3D.

// quaternion → ângulos

Do quaternion derivam-se os ângulos de Euler: roll (tombamento) · pitch (inclinação) · yaw (rotação horizontal).

→ cérebro mecanicista — valores embutidos, sem improviso.

→ percepção — análogo ao sistema vestibular.
Captam aceleração e rotação assim como o labirinto do ouvido interno capta equilíbrio e movimento da cabeça.

→ condicionamento da escuta e percepção corporal.
O código define o que o sistema "presta atenção" — assim como treinamento corporal afina a percepção cinestésica.
