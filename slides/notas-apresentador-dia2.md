## Ciclo 1 — LED interno

**Slide:** "LED interno → Arduino IDE"

1. Mostrar a placa fisicamente. Apontar onde fica o LED embutido (pino 2 / GPIO2).
2. Abrir o Arduino IDE ao vivo.
3. Apresentar o repertório básico:
   - `setup()` — roda uma vez na inicialização
   - `loop()` — roda para sempre em repetição
   - `pinMode(pino, OUTPUT)` — configura o pino como saída
   - `digitalWrite(pino, HIGH/LOW)` — liga ou desliga
   - `delay(ms)` — pausa em milissegundos
4. Escrever ao vivo o código de piscar o LED no pino 2.
5. Compilar e gravar. Mostrar o LED piscando.
6. Pedir para todas replicarem.

**Desafio:** Mudar os valores de `delay()` para o LED piscar mais rápido ou mais devagar.
