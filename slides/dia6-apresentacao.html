// firmware — gravar no Giromin

Firmware do Giromin

// baixar

Releases do repositório:
github.com/jtragtenberg/oficina-programacao-danca/releases

Arquivo: giromin-firmware.zip

// configurar

Abrir giromin-firmware.ino e editar:

ssid = "NOME_DA_REDE"
password = "SENHA_DA_REDE"
ID = 1  ← número do Giromin
destIP(192, 168, x, x)  ← IP do computador

// calibração

Segure o botão ao ligar → aguarda 3s → calibra → salva na EEPROM.
Fazer com o Giromin na posição de repouso (ex: pulso relaxado horizontal).

// mensagens OSC enviadas

/giromin/ID/a  → acelerômetro x, y, z (−1 a 1)
/giromin/ID/g  → giroscópio x, y, z (−1 a 1)
/giromin/ID/b1 → botão: 1 ao apertar, 0 ao soltar

// setup — paleta de objetos

Instalar a Paleta Giromin

// baixar

Releases:
github.com/jtragtenberg/oficina-programacao-danca/releases

Arquivo: exemplos-pd.zip
Dentro: exemplos-pd/giromin.pdpalette

// instalar — macOS

Copiar giromin.pdpalette para:
~/Documents/PlugData/Palettes/

// instalar — Windows

Copiar para:
%APPDATA%\PlugData\Palettes\

// instalar — Linux

Copiar para:
~/.local/share/PlugData/Palettes/

// ativar

Reiniciar o PlugData.
A paleta Giromin aparece na barra lateral esquerda com blocos prontos para usar.

// exemplos — pasta de patches

exemplos-pd/

// estrutura

Cada ciclo tem três arquivos:

a-base-*.pd   ← ponto de partida (em branco)
b-aula-*.pd   ← versão da aula com desafio
c-desafio-*.pd ← resolução do desafio
